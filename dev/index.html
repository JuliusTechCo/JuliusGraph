How to start

There are different ways to use the tutorials:

  • Reading the html version of the tutorials. This is the recommended way if you want rapid access to the material with no setup steps. Simply click in one of the links in the Contents section.
  • Running the Jupyter notebooks locally. A working installation of Julia in the system is required. See instructions in the How to run the notebooks locally section. This is the recommended way to follow the tutorials if you want to run the code and inspect the generated results.

How to run the notebooks locally

Clone the repository

$ git clone git@github.com:JuliusTechCo/Tutorials.git

Move into the folder and open a Julius REPL setting the current folder as the project environment.

$ cd Tutorials
$ julius --project=.

Instantiate the environment. This will automatically download all required packages.

# Type ] to enter in pkg mode
(Tutorials) pkg> instantiate

Build the notebooks by either doing

(Tutorials) pkg> build

or

# Type Ctrl+C to get back to command mode
julia> include("deps/build.jl")

Open the notebooks

julia> using IJulia
julia> notebook(dir=pwd())

This will open a browser window. Navigate to the notebooks folder and open the tutorial you want. Enjoy!
